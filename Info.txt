Software: JDK (Paquete de Software) JAVA DEVELOPMENT KIT

Java no es un lenguaje ni es de Oracle, es una especificación. Es más que un lenguajee
Compilar: crear un ejecutable
Java tiene la ventaja que cuando se compila en una plataforma puede utilizarse en otras plataformas.
Java es lenguaje libre pero el compilador es privativo (desde que lo compró Oracle). De todas formas
se pueden encontrar compiladores libres para usarlo.

Comunidad: JCP Java Community Process
Es un consorcio formado por empresas (Oracle, IBM, RedHat, Amazon) que dictaminan cómo tiene que ser el lenguaje.
Java es corporativo de por sí, no es para pequeñas empresas o emprendimientos.
